# Emotion-Aware-Sarcasm-Detection-in-Multimodal-
HCI-EASD (Hierarchical Cross-Modal Incongruity and Emotion-Aware Sarcasm Detection) This model four key modules: multimodal feature extraction using BERT and ResNet-50, emotion-aware processing with incongruity correlation analysis, cross-modal incongruity detection through multi-head attention, and hierarchical classification with loss functions.
